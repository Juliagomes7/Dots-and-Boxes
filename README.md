# Dots and Boxes

Recriação do jogo **Dots and Boxes** desenvolvida em Java como parte do primeiro contato com a linguagem.

**Objetivo**

O objetivo deste projeto foi praticar os conceitos iniciais da linguagem Java, como
estrutura de classes, lógica de programação, controle de fluxo e organização básica de código.

**Tecnologias utilizadas**

- Java

**Funcionalidades**

- Implementação da lógica básica do jogo Dots and Boxes
- Controle de turnos dos jogadores
- Contabilização de pontos
- Encerramento da partida
